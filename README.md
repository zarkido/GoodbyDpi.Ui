# GoodbyeDPI GUI (C#)

This project is a graphical user interface (GUI) built for a C#-based implementation inspired by GoodbyeDPI.

It provides a simple way to use DPI bypass techniques without requiring command-line usage.

## Features

* Minimal and simple interface
* Start / stop control
* No command-line interaction required
